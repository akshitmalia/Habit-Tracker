#  Habit Tracker App

 * 🌐 Live Demo:  https://akshitmalia.github.io/Habit-Tracker/

A simple and interactive habit tracking application that helps users build consistency by adding, tracking, and monitoring daily habits with progress visualization.

---

## 🚀 Features

* ➕ Add new habits
* ✅ Mark habits as completed
* ❌ Delete habits
* 🔄 Reset all habits
* 📊 Dynamic progress bar (completion %)
* 💾 Data persistence using localStorage
* 🎯 Real-time UI updates

---

## 🛠️ Tech Stack

* HTML
* CSS (Bootstrap 5)
* JavaScript (Vanilla JS)
* LocalStorage

## ⚙️ Installation & Setup

```bash
git clone https://github.com/akshitmalia/Habit-Tracker.git
cd Habit-Tracker
```

* Open `index.html` in your browser

---

## 📌 How It Works

1. User enters a habit and clicks **Add**
2. Habit appears in the list
3. User can:

   * Mark it as completed
   * Delete it
4. Progress bar updates automatically
5. Data is saved in localStorage (persists after refresh)

---

## 📊 Progress Logic

* Progress = (Completed Habits / Total Habits) × 100
* Automatically updates on every action
